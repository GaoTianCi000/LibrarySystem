这是一个基于SSH（Struts2+Spring+Hibernate）框架整合的图书管管理系统(MVC架构)
前台主要功能：登录注册 按照个人查询已经借阅的书籍 借阅书籍   查询书籍  归还书籍等
后台主要功能：对前台前所有功能进行管理


关于数据库：
	根据SSH整合的配置文件中 hibernate部分  加上运行hibernate后自动生成数据库的配置即可 非常简单 不会的在github给我留言
	或者：
		根据实体类来创建数据库

关于jar包：
	Struts 2.1.6 + Spring2.5.6 + Hibernate3.3 网上下载即可
